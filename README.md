## ⚡ Featured Project — Sparx Build Tool

> *"Why use Webpack or Vite when you can build your own?"*

**[Sparx](https://github.com/Avinash-1994/sparx)** is a production-grade JS/TS build tool I authored from scratch — powered by **SWC (Rust)** and **LightningCSS**.

<table>
<tr>
<td width="55%">

| Feature | Detail |
|---------|--------|
| ⚡ **Rust-native core** | SWC + LightningCSS, faster than esbuild |
| 🔥 **Sub-100ms HMR** | p50: 12ms · p95: 14ms |
| 🏗️ **19 meta-frameworks** | Next.js, Nuxt, Astro, Remix, SvelteKit… |
| 📦 **Module Federation** | Native MFE host/remote orchestration |
| 🔒 **Security pipeline** | CVE scan, CSP, SRI, SBOM |
| 💾 **Smart cache** | SQLite WAL incremental build cache |
| 🔌 **11 official plugins** | env, pwa, icons, svg, legacy, image… |

</td>
<td width="45%" align="center">

```bash
npm install -g sparx

sparx create my-app \
  --framework react --ts

sparx dev
# → localhost:5173  < 100ms
```
